# skills
My own skills for AI agents

Some of these skills are from [mattpocock/skills](https://github.com/mattpocock/skills), with some customization to make them works better for me. 

## Why customize

I can't afford to pay several hundred dollars to LLM providers (My living costs are less than $200 a month, excluding rent!). Instead, I use local LLMs like QWen3.6-35B-A3B and QWen3.6-27B. They work, but they are much less clever than Claude Fable or GPT-5.6. Therefore, mattpocock's skills are not always suitable for my needs.
